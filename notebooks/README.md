# Notebooks

- `latentdriver_assets_colab.ipynb`: clone/sync repo, mount Drive, patch upstream, download public evaluation checkpoints.
- `latentdriver_preprocess_val_colab.ipynb`: prepare smoke or full validation preprocessing artifacts from WOMD validation data.
- `latentdriver_public_eval_colab.ipynb`: evaluate the public checkpoints under standardized reactive/non-reactive Waymax tiers and compare against the paper table.
- `latentdriver_visualize_colab.ipynb`: run one smoke visualization job and surface the generated MP4/PDF artifact.
